# Railway Ticket Booking System

## Overview

This is a simple Java program for a railway ticket booking system.
The program allows users to display train information, check seat availability, book tickets, cancel tickets, and display booked tickets with amounts.

## How to Compile and Run

1. Make sure you have Java installed on your machine.
2. Open a terminal or command prompt.
3. Navigate to the directory containing the Java files.
4. Compile the program: `javac RailwayTicketBooking.java`
5. Run the program: `java RailwayTicketBooking`

## Usage

Follow the on-screen instructions to interact with the Railway Reservation System.
The program provides a menu-driven interface for various actions such as displaying train information, checking seat availability, booking tickets, canceling tickets, and displaying booked tickets with amounts.
